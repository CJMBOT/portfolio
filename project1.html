Project 1: Spirit Animal Randomiser

This was the result of my 'Mixed Messages' project set by Codecademy that challenged me to use Javascript in order to create a message that would change each time you executed it.

The script contains a few arrays from which a random item is chosen to populate certain parts of the final message. This message tells you what your spirit animal is today, as well as where you get your strength from, what causes your weakness and ends with a little nugget of wisdom.

I threw in some extra sections to the code for example with the animal, there are a couple of lines that check whether the first letter of the randomly chosen animal is a vowel or consonant. When it's a consonant it ensures the preceding word is "a" and when it's a vowel the preceding word is "an". For example this will correctly say "a pig" or "an eagle".

I also included two different arrays of quotes; one serious and one humorous. I used an extra random number generator to decide which list was picked from for the final output.

With 80 possible animals, 80 strengths, 46 weakness and a combined 39 quotes of advice, this randomised message has 11,481,600 possible combinations.
